# Builder Artifact Placeholder

This directory is reserved for builder-owned bootstrap, publication, and intermediate artifacts.

- `bootstrap.json` remains the quarantined fixture bootstrap surface used by current viewer and review harnesses.
- `opening-table/manifest.json` and `endgame-table/manifest.json` are reserved for project-owned N11d table publication manifests and their builder-written shards.
- `web-corpus.json` is reserved for the combined N11d manifest that records the published table assets, shared contract metadata, and deterministic hashes.
